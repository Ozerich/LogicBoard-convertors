Установка
Введение.
Данная инструкция предназначена для запуска/установки конвертора с TWSF 1.7.1 на LogicBoard.

Входе конвертации будут стёры данные от учётной записи администратора, которая была создана во время установки форцма.

Инструкция:
1) Установите форум LogicBoard (инструкция в дистрибутиве форума)

2) Зайдите в центр управления -> настройки -> Загрузка файлов.

2.1) Добавьте новую настройку: (поля, которые нужно заполнить указаны ниже)
Название: "Ссылка на файл"
Описание: "Ссылка на прикреплённый файл с TWSF"
Тип: "Одна строка"
Значение (ключ) по умолчанию: "Введите ссылку на папку с файлами. Например: http://site.ru/uploads/forum/files/ (где site.ru - сайт с движком DLE)"
Ключ: "upload_convert"

2.2) Добавьте новую настройку: (поля, которые нужно заполнить указаны ниже)
Название: "Ссылка на картинку"
Описание: "Ссылка на прикреплённую картинку с TWSF"
Тип: "Одна строка"
Значение (ключ) по умолчанию: "Введите ссылку на папку с картинками. Например: http://site.ru/uploads/forum/posts/ (где site.ru - сайт с движком DLE)"
Ключ: "upload_convert_img"

3) Загрузите все файлы из папки upload в корневую папку форума

4) Запустите конвертор, введя в адресную строку браузера http://адрес_форума/twsf.php

5) Введите данные от БД TWSF и данные от БД LogicBoard

6) Обязательно удалите загруженные файлы конвертора!
